# FoodHub - Food Ordering Website

## Project Description

FoodHub is a responsive Food Ordering Website developed using HTML, Bootstrap, and JavaScript. The project demonstrates DevOps practices using GitHub, Jenkins, Docker, and Ansible.

## Features

* Home Page
* Menu Page
* Shopping Cart
* Contact Page
* Add to Cart Functionality
* Responsive Design

## Technologies Used

* HTML5
* Bootstrap 5
* JavaScript
* GitHub
* Jenkins
* Docker
* Ansible

## DevOps Workflow

GitHub → Jenkins → Docker → Ansible → Deployment

## Docker Commands

Build Image

```bash
docker build -t food-ordering-app .
```

Run Container

```bash
docker run -d --name food-ordering-container -p 8080:80 food-ordering-app
```

## Website URL

http://localhost:8080
